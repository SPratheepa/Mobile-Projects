# NeoLoop Frontend

## Overview

NeoLoop is an AI-powered Escape Room Platform where creators can build immersive escape room experiences and players can explore, solve puzzles, and compete on leaderboards.

This repository contains the Frontend Prototype built using:

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* REST API Integration

# Running Frontend

Install VS Code Live Server Extension.

Open frontend folder.

Right Click:

index.html

Select:

Open with Live Server

Application starts at:

http://127.0.0.1:5500

# Backend Requirement

Backend must be running before frontend.

Backend URL:

http://127.0.0.1:5000

Swagger:

http://127.0.0.1:5000/apidocs
